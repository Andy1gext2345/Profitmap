how to deploy the apps script
  1. Open your Google Sheet → click Extensions → Apps Script
  2. Delete any existing code and paste the entire contents of SterlingOak_AppScript.gs
  3. Click Save (disk icon), then click Deploy → New deployment
  4. Set type to Web app · Execute as: Me · Access: Anyone
  5. Click Deploy → copy the Web App URL
  6. Paste that URL in the connection box above and click Save & Connect
